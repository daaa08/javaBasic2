#### 클래스
- 절차적으로 가다가 일정한 패턴이 나오면 클래스로 묶음

``` java
        String Student1Name = "홍길동";
		int student1Age = 15;

		String Student2Name = "이순신";
		int student2Age = 25;

		String Student3Name = "자바다";		
		int student3Age = 35;
```

Student1Name, student1Age 가 중복되므로...


``` java
class Class{
	String Name;   
	int age;
}
```
위 처럼 묶기 가능함.
